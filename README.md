# 課題LINE風チャットアプリ
・タイトル　ちょこっとチャットしよっと  


## ①課題内容（どんな作品か）
・firebaseを使うということ以外はうまく作ることが出来なかった
リアルタイムチャットアプリです。

## ②工夫した点・こだわった点
・チャットのboxの影をつけたところ。

・ワールドカップの日本代表のユニフォームから影響された
色の配色→青系で白っぽく見えているところも青系の
白い感じで全体的に青にこだわったところ。

## ③難しかった点・次回トライしたいこと(又は機能)
・firebaseとの連携が全然うまくできなくてずーっとnullが
続いていて、夢にまでnullが出てくるくらい困りました。

・firebaseを使って画像保存のアプリも少し作ってみたんですが、
それを使った作品が思い浮かばず、途中になってしまったものを
何かしらで再利用したい。
